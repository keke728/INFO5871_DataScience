Question 4: Assessing seasonal variation¶

Using the $\chi^2$ method, determine if this observed seasonal variation is statistically significant.

What is the null hypothesis?¶

H0: The observed seasonal variation is a random sample, there is no significant seasonal variation between the observed value and the expected value, and the observed variation reflects natural sampling fluctuation.

Call the chisquare function to do the calculation¶

From the result, print the $\chi^2$ statistic. res.statistic and the p-value res.pvalue.

In [20]:
total_flights_2006 = total_flights_2006.reset_index(drop=True)
expected = [47253.5, 47253.5, 47253.5, 47253.5, 47253.5, 47253.5, 47253.5, 47253.5, 47253.5, 47253.5, 47253.5, 47253.5]
observed = total_flights_2006['Total Flights'].tolist()
In [21]:
st.chisquare(observed, expected)
Out[21]:
Power_divergenceResult(statistic=2125.6239431999747, pvalue=0.0)

Conclusion¶

The $\chi^2$ statistic calculated value is way bigger than the $\chi^2$ critical value, and p=0, so we reject the null hypothesis, and conclude that there is a significant seasonal variation between the observed value and the expected value.

Create a projected 2006 column¶

This is the number of flights for each month if the distribution was the same as 2006. For example, January projected = January 2006 / Total for 2006 * Total of 2018. That way the count for 2018 and the projected count are the same, but the project column has the proportional distribution of 2006.

We need to do this because the $\chi^2$ statistic depends on the counts and if the totals are different, the results aren't meaningful.

In [27]:

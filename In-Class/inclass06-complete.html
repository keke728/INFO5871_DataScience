Question 1: American vs Delta¶

What proportion of months did American have more passengers than Delta? Based on a sample of the data, can we estimate accurately?

We know the "real" answer, because we have all the data, so this is just an exercise.

Prepare¶

  • Identify the parameter of interest
  • List hypotheses
  • Identify the signifance level
  • Identify $\hat{p}$ and n.
  • Parameter of interest: proportion of months in which American > Delta
  • H0: The fraction is 50%. No difference in the airlines.
  • H1: There is a significant difference
  • Significance level 95% $\alpha$ = 0.05
  • n = 100

Computing $\hat{p}$

In [8]:
american_more = pass_df['American'] > pass_df['Delta']
n = 100
american_sam = american_more.sample(n)
p_hat = american_sam.sum() / n

Check.¶

  • Verify conditions to ensure $\hat{p} is nearly normal under H0. For one-proportion hypothesis tests, use the null value to check the success-failure condition.
In [9]:
print (n*p_hat > 10)
print (n*(1-p_hat) > 10)

True
True

Calculate.¶

  • If the conditions hold, compute the standard error, again using p0,
  • compute the Z-score,
  • identify the p-value.
In [10]:
se = np.sqrt(p_hat*(1-p_hat)/n)
z = (0.5 - p_hat)/se
print(z)

-2.4722569302909876
In [11]:
p = 2 * st.norm.cdf(z)
print (p)

0.013426298293326975

Conclude.¶

Evaluate the hypothesis test by comparing the p-value to $\alpha$, and provide a conclusion in the context of the problem.

$p$ value is much less than that $\alpha$. Therefore, we conclude that American has more flights than Delta, and this effect is very significant.

Melt the data¶

Use pandas's melt function on your cleaned passengers_df to make it tidier. Note that you'll likely need to use .reset_index() on passengers_df to turn the "Month" index back into a column. Rename the columns so they're more interpretable too.

Your melted data should look something like this:

Month Airline Count
1995-02-01 American 5833
1995-03-01 American 278811
1995-04-01 American 184229
1995-05-01 American 187990
1995-06-01 American 229348
In [14]:
pass_df = pass_df.reset_index()
pass_melt = pd.melt(pass_df, id_vars=['Month'])
pass_melt.columns = ['Month', 'Airline', 'Count']

Create a new column with different ranges of cp¶

We could do this with cut but this is a different way. We will designate Pokemon with CP less than 100 as 'weak'; those with CP between 100 and 200 as 'moderate'; 200 to 300 as 'strong' and above 300 as 'very strong'.

In [31]:
# This creates the column and gives all values as 'weak'. That's OK because we will next overwrite the values
pidg['StrengthLevel'] = 'weak'
moderate_filter = (pidg.cp > 100) & (pidg.cp <= 200)
pidg.loc[moderate_filter, 'StrengthLevel'] = 'moderate'
strong_filter = (pidg.cp > 200) & (pidg.cp <= 300)
pidg.loc[strong_filter, 'StrengthLevel'] = 'strong'
vstrong_filter = (pidg.cp > 300)
pidg.loc[vstrong_filter, 'StrengthLevel'] = 'very strong'
